# 🚀 FIX Log Explainer

AI-powered FIX, MT4, MT5, Bridge & Liquidity Log Analyzer for Forex Brokers.

---

## Features

- 📂 Upload FIX Log Files
- 📋 Paste Raw FIX Logs
- 🤖 AI-Powered Incident Summary
- 📊 Order Timeline
- 🔍 Root Cause Analysis
- ⚠️ Issue Detection
- 📈 Execution Analytics
- 📄 Client-Friendly Reports
- 📥 Export CSV Report

---

## Supported Platforms

- FIX 4.2
- FIX 4.4
- FIX 5.0
- MT4
- MT5
- Centroid
- OneZero
- PrimeXM
- FXCubic

---

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
